#### 解决什么问题

批量下载晓黑板班级作业

#### 使用

- windows 下直接运行
  下载 bin 中文件并执行
- linux 等其他平台请下载代码自行编译

- 输出
  - $yn_local_dir/cache/download/ 下载作业
